# BagUI Studio

BagUI Studio is a VS Code extension for browsing, previewing, searching, and installing BagUI components from the shared registry without leaving the editor.

## Features

- Browse components by category in the Activity Bar
- Search components instantly with Quick Pick
- Open rich component previews in a webview
- Install components using the BagUI CLI
- Save favorites globally in VS Code
- Refresh the registry on demand

## Development

```bash
npm install
npm run build
```
